# BIZENIX

A modern student entrepreneurship platform built with Next.js, React and Tailwind CSS.

## Features

* Modern startup landing page
* Founder profile page
* Responsive design
* Smooth scrolling experience
* Animated counters and interactions
* SEO optimized
* Custom branding and favicon
